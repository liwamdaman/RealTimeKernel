# rtos
Real-Time Kernel
By Kimathi and Will

Hardware:
- ARM Cortex-M3 processor
- LPC1768 MCU
- Keil MCB 1700 evaluation board (uVision5 IDE)

Features:
- Fixed-priority preemptive scheduling of up to 6 tasks, managed with TCBs
- Mutexes with owner test on release and priority inheritance
- Blocking semaphores
